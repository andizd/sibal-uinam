# sibal-uinam
SIBAL (Sistem Informasi Barang Hilang) adalah aplikasi web sederhana yang dibuat untuk membantu mahasiswa menemukan dan melaporkan barang hilang di area kampus. Website ini menjadi pusat informasi barang hilang yang mudah diakses oleh seluruh civitas kampus.  Proyek ini dibuat sebagai tugas akhir mata kuliah Pemrograman Web.
